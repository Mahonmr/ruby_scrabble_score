#Scrabble Score

###By: Krystan Menne and Mike Mahoney

Scrabble is a sample ruby application.

A user enters a word and the program returns a scrabble score for that word. Letters are scored as follows:

* A, E, I, O, U, L, N, R, S, T       1
* D, G                               2
* B, C, M, P                         3
* F, H, V, W, Y                      4
* K                                  5
* J, X                               8
* Q, Z                               10
